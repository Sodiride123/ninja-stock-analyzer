  • ✓ All US-incorporated public companies on NYSE / NASDAQ — large-caps like GOOGL, NVDA, MSFT, mid-caps like CROX, small-caps like CTRN, etc. — as long as they have at least one recent 10-Q or 10-K filing on SEC EDGAR.

Rule of thumb: if the company is a US domestic filer on SEC EDGAR, it works. If it's a foreign issuer on ADRs, it won't.

📖 Stock Analyst — Help Guide

🚀 Getting Started

Stock Analyst is an AI-powered research tool that automatically analyzes publicly traded companies using SEC filings, financial data, and AI-driven insights.

📊 Ways to Use the App

1. Web Dashboard (this page)

  • Run Analysis: Type a stock ticker (e.g. AAPL, NVDA, TSLA) in the input field and click ▶ Run Analysis.
  • Auto-Select: Click ⚡ Auto-Select to let the AI pick an interesting company to analyze.
  • My Stocks: Previously analyzed stocks appear in the bar at the top. Click any chip to reload that analysis instantly.
  • Rerun / Delete: Use the ↻ and ✕ buttons next to each stock chip to rerun or delete an analysis.
  • Direct URL: Share or bookmark a URL like #NVDA to jump directly to a stock's analysis.

2. Command Line (CLI)

Run analysis from the terminal using the CLI script:

python run_analysis.py AAPL              # Analyze (cleans existing data first)
python run_analysis.py AAPL MSFT GOOG   # Analyze multiple tickers
python run_analysis.py --all             # Re-run ALL existing analyses
python run_analysis.py --keep AAPL       # Analyze without deleting existing data
python run_analysis.py --list            # List all past analyses

📑 Analysis Pipeline (12 Steps)

  1. Select Company — Identifies the company from the ticker symbol
  2. Research News — Gathers recent news and market context
  3. Download Reports — Fetches 10-K and 10-Q filings from SEC EDGAR
  4. Extract Numbers — Pulls key financial metrics from filings
  5. Extract Goals — Identifies management goals and forward guidance
  6. Analyze Tone — AI sentiment analysis of filing language
  7. Analyze Price — OHLC price data with technical analysis
  8. Get Logo — Fetches the company's official logo
  9. Compare Reports — Quarter-over-quarter comparison
  10. Generate Report — Creates a comprehensive HTML/PDF report
  11. Ten-Point Analysis — Yay/Nay bullet-point summary
  12. Animate — Generates an animated OHLC chart video (shown at top of Overview)

📈 Dashboard Tabs

  • Overview — Summary with key metrics, company logo, and OHLC animation video
  • News — Recent news and market context
  • Numbers — Extracted financial data from latest and prior filings
  • Goals — Management guidance and strategic goals
  • Tone — Sentiment analysis of SEC filing language
  • Price — Interactive OHLC chart with volume, stats, and AI analysis
  • Comparison — Side-by-side comparison of two filing periods
  • Report — Full HTML/PDF research report
  • Bullets — Quick yay/nay ten-point summary
  • Chat — Ask Claude questions about the loaded stock
  • Logs — Pipeline execution logs and run history

⚙️ Tips

  • Analysis typically takes 3–15 minutes depending on filing size.
  • Large filings (70+ pages) may cause some steps to time out — the pipeline continues past failures.
  • All data is persisted to disk — reload the page anytime without losing progress.
  • The pipeline steps run in parallel where possible for faster results.
  • Re-run individual steps by right-clicking any tab (except Overview). This lets you refresh specific parts of the analysis without re-running everything. Tabs show status indicators: 🟢 done, 🟡 running/partial, 🔴 error.
